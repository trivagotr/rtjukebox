# Feature Research: RT Jukebox

## Verified v1 Capabilities

- Registered user creation and authenticated profile lookup.
- Guest user creation with guest role persistence.
- Gamification games endpoint access for authenticated users.
- Jukebox song search contract from mobile to backend.
- Mobile RSS podcast parsing without vulnerable RSS parser dependency.
- Kiosk Spotify player tests on upgraded Vitest dependency set.
- Android debug APK generation for mobile and automotive variants.

## Deferred Capabilities

- Production deployment and smoke testing.
- Android release signing and store distribution.
- Manual physical-device QA.
